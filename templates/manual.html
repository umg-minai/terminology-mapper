Step 2: Entering Codes and Display Texts

For each concept you identify, you need to provide two pieces of information:

  • Code: The unique identifier from the vocabulary system (e.g., SNOMED CT, ICD-10, LOINC)
  • Display Text: The human-readable description that corresponds to the code

Example:

  • Code: I10
  • Display Text: Essential (primary) hypertension
  • Vocabulary: ICD-10

üí° When No Code Exists - Proposing New Concepts

Important: Only use the "No Code Found - Propose New Concept" option when you have thoroughly searched and cannot find an existing code in any vocabulary.

What this means:

  • You've searched all relevant vocabularies (SNOMED CT, ICD-10, LOINC, etc.)
  • No existing code adequately represents the term
  • You're proposing that a new concept should be created

How to propose a new concept:

  1. Click the "No Code Found - Propose New Concept" button
  2. Enter only a display text (description) for the proposed concept
  3. Leave the code field empty - this indicates it's a new proposal

Note: Proposed new concepts will be marked as vocabulary "NEW_CONCEPT" and will be reviewed separately.
